# voceLevaMvp
**Requeriments**
nodejs 5.x +

**scripts**
To install node dependencies:
```bash
$ npm run start
```

To build:
```bash
$ npm run build
```

To build and start server:
```bash
$ npm run dev
```
A local browserSync server will start on localhost, a port must be set on /tools/config.js, default is 3000
